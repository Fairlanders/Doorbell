# Doorbell
A smart doorbell which plays random songs.

## Requirements

Things that you need:

```
1-A Raspberry Pi or Arduino Uno
2-Two buttons
3-Two 100nF transistors.
4-A solderer and a long cable
5-Ability to google
```
